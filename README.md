# Backend Developer Assessment

## Features
- User Registration & Login
- JWT Authentication
- Protected APIs
- Order Management

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt

## API Endpoints
- POST /api/register
- POST /api/login
- GET /api/profile (Protected)
- POST /api/order (Protected)
- GET /api/orders (Protected)

## How to Run
1. npm install
2. Create `.env` file
3. npm run dev
